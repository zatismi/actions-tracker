About Actions Tracker

This app is intended for learning purposes and includes the following concepts

  • Variables & constants → storing state (historyArray, playClicked, interval).
  • DOM selection & manipulation → getElementById, getElementsByClassName, .innerHTML, .classList.
  • Event handling → triggered by onclick, oninput.
  • Functions → modular logic (record, play, pause, etc.).
  • Conditionals → controlling flow with if.
  • Loops / Iteration → setInterval with incrementing index.
  • Arrays → storing text history (historyArray).
  • Type conversion → parseInt() for slider value.
  • State management → toggling playClicked, tracking current index.
  • Timers → setInterval and clearInterval.
  • UI updates → syncing text area, preview box, slider, button states.
  • Class-based animations → shown/hidden toggle for transitions.
  • Debugging / Logging → console.log.
